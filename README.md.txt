# Smart Data Pipeline for E-Commerce Analytics

## 📌 Project Description
This project implements an end-to-end ETL (Extract, Transform, Load) pipeline using Python and MySQL. The pipeline processes raw CSV data, cleans it, stores it in a database, and performs SQL-based analysis to generate insights.

---

## 🚀 Features
- Data cleaning using Python (Pandas)
- Data loading into MySQL database
- SQL-based data analysis
- Structured and modular project design

---

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- MySQL
- SQLAlchemy
- PyMySQL
